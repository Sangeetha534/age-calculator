# 🎂 Age Calculator

A responsive and user-friendly Age Calculator built using **HTML, CSS, JavaScript, and Luxon**.

This application allows users to select their birthdate using a JavaScript datepicker and calculates their exact age in **years, months, and days**.

---

## 🚀 Features

- 📅 JavaScript Datepicker (Flatpickr)
- 🧮 Accurate age calculation using Luxon
- ✅ Birthdate validation
- 📱 Fully responsive design
- 🎨 Clean and simple UI
- ⚡ Instant result display without page reload

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Luxon (Date/Time calculations)
- Flatpickr (Datepicker library)
